Parameters
$prefDirThe preferred directory path to use.
$altDirThe alternative directory path.
$commonPathThe common path.

This function tries to return the path to the file $commonPath searching first in the preferred directory $prefDir and if not found in this one, it searches into the alternative directory $altDir. This function does not check if $commonPath file really exists in $altDir directory.
